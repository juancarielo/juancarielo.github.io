At **VitaminaWeb**, working as a **Developer**, I built and maintained a virtual secretary system using **CodeIgniter** and **MySQL**, integrated with **Asterisk** for VoIP telephony.

The system allowed calls to be received, routed, and logged automatically, bridging the web application with the telephony server so call data, contacts, and schedules stayed in sync in real time.

It was my first time integrating a traditional web stack with VoIP infrastructure, going beyond typical CRUD applications.
